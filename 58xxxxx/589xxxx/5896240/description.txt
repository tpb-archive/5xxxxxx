This is Lubntu 10.10 Maverick Meerkat, light version of the Ubuntu Linux.
Most suitable for netbooks or weak or old computers.
Very fast - gets booted in 21 seconds from grub to working state on an Asus 1005HA netbook.
Possible to use as Live Image.

Installation:

Burn to CD, or, better yet,
mount it on USB Stick or SD Card with PendriveLinux program (search for it)
follow the instructions of the installer.

U can install it alongside Windows, so you dont have to format the hard drive and destroy the Windows installation.

The OS is open source, and the developers encourage to download it from torrents, to reduce load on their servers
